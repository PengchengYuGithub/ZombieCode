Input section

Click the "gridSize" button to get the size of the grid.

"gridSize: default 4"

Click the "zombiePosition" button to get the position of the zombie.

"zombiePosition: default x: 3, y: 1"

Click the "creaturesPosition" button to get the positions of the creatures.

"creaturesPosition: default
x: 0, y: 1
x: 1, y: 2
x: 1, y: 1"

Click the "commands" button to get the positions of the creatures.

"commands: default RDRU"

(if not set, will follow default parameters)
